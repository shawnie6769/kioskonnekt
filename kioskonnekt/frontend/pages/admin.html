Admin Dashboard Help

Overview: This dashboard allows you to monitor and manage university admissions applications.

Key Features:

  • Applicants Table: View all applicants with their status, program, and documents
  • Statistics: See real-time stats on total applications, completions, and daily activity
  • Filtering: Filter applicants by status (All, Completed, In Progress)
  • Sorting: Click column headers to sort by name, status, or date
  • Applicant Details: Click on an applicant row to view full details and documents

Actions:

  • Edit: Modify applicant information if needed
  • View Documents: Click document links to view scanned images
  • Refresh: Use the refresh button to reload latest data

For technical support, contact the IT department.
